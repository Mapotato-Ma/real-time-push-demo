# real-time-push-demo

# 前提

本机已预装 node 与 npm

# 启动后端服务：

1. 启动接口服务：在 my-nest-project 目录下打开终端，输入 powershell 命令

```js
  npm install

  npm run start
```

2. 启动 websocket 服务：在 node 目录下打开终端，输入 powershell 命令

```js
  npm install

  node app.js
```

# 启动前端服务：

在 share-demo 目录下打开终端，输入 powershell 命令

```js
  npm install

  npm run dev
```

浏览器中打开前端服务启动对应网址即可看到效果
